## About The Project

[![Product Name Screen Shot][product-screenshot]](https://example.com)

I wanted to create a cool LED Controller App.

Here's why:
* This is something that has already been done. I want to to see if I can do it - for better or for worse.
* I'd like to put my own spin on the idea.
* I think this will be a cool resume project to try.

My main goal is to make an app that can interface with different types of arduino based, or maybe non-arduino based, boards that can control a set of LEDS.
At some point I would like the LED's to work with whichever effects i'd like, and on multiple platforms.
